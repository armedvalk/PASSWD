# PASSWD
PASSWD is an Open Source password generator focused on making secure passwords, without compromisng integreity.
--------
# How to use?
To use PASSWD, simply run the .exe file and the application will guide you through the rest.
---------
# Why can we trust you?
This app is such as simple app, it is just a python file that generates a random string of random charecters and lets you copy it, no security threat there!
